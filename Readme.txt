Project: Movie Management App

This program will read in a .txt file that is filled with a list of movies.
You then have 4 options:

1 	Add New Movie and Save 
2 	Generate List of Movies Released in a Year 
3 	Generate List of Random Movies 
4 	Save & Exit 

There is a .jar file included that you can run the program from
